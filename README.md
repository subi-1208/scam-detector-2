# Job Scam Detector

가짜 해외 취업 제안, 고수익 아르바이트 광고, 수상한 채용 사이트를 분석해서 취업사기 위험도를 점수 형태로 보여주는 웹서비스다.

## 한 줄 목표

사용자가 입력한 텍스트나 URL이 취업사기일 가능성을 `0~100` 점수와 근거 설명으로 빠르게 이해할 수 있게 만든다.

## 핵심 문제 정의

- 해외 고수익 취업 제안
- 메신저 기반 비정상 채용 유도
- 회사 정보가 불명확한 채용 공고
- 캄보디아 등 동남아 연계 취업사기·감금형 스캠 유사 패턴

중요한 전제:
이 서비스는 범죄 여부를 확정하는 판정기가 아니라, 위험도 예측과 피해 예방 안내를 제공하는 참고 도구다.

## 주요 사용자

- 해외 취업 공고를 검토하는 일반 사용자
- 부업·아르바이트 제안을 받은 사용자
- 수상한 채용 사이트를 점검하려는 사용자
- 가족이나 지인의 제안을 대신 확인해 주는 사용자

## 핵심 입력

### 1. 텍스트 분석

- 채용 공고
- 메신저 대화
- 문자
- 이메일

### 2. URL 분석

- 채용 사이트 주소
- 지원 페이지 주소

## 반드시 보여줘야 할 결과

- 위험 점수: `0~100`
- 위험 등급: `낮음 / 주의 / 위험 / 매우 위험`
- 주요 경고 사유 3개 이상
- 세부 위험 요인별 부분 점수
- 사용자 행동 권고
- 참고용 예측이라는 고지

## 주요 위험 신호

- 과도하게 높은 급여
- 업무 내용 불명확
- 빠른 출국 또는 즉시 근무 요구
- 여권, 항공권, 숙소 제공 강조
- 회사명, 주소, 사업자 정보 부족
- 텔레그램, 왓츠앱, 라인 등 외부 메신저 유도
- 수수료, 보증금, 선입금 요구
- 비정상적 문체 또는 번역투 표현
- 캄보디아, 카지노, 동남아 온라인 업무 조합 등 고위험 패턴

## URL 분석에서 특히 확인할 것

- HTTPS 적용 여부
- 회사 소개, 주소, 연락처 존재 여부
- 채용 페이지의 구체성
- 공격적인 지원 유도 문구
- 외부 메신저 연결 유도
- 공개 평판 연계 가능성
- 접근 실패 시 실패 사유 안내

## MVP 범위

- 텍스트 입력 분석
- URL 입력 분석
- 위험 점수 산출
- 근거 3개 이상 설명
- 결과 저장
- 관리자 룰셋 수정

## 현재 구현 방향

- Next.js App Router 기반
- 초기 엔진은 `룰 기반 + 간단한 휴리스틱`
- 분석 결과는 설명 가능성을 우선한다
- 모바일에서도 쉽게 입력하고 확인할 수 있어야 한다
- 결과 저장은 현재 로컬 파일 기반으로 단순화한다
- 관리자 룰셋은 빠르게 수정 가능해야 한다

## UX 원칙

- 공포 조장보다 이해 가능한 설명을 우선한다
- 점수만 던지지 말고 이유를 같이 보여준다
- 고위험일수록 송금 금지, 여권 사본 주의, 가족 확인, 공공기관 문의를 함께 안내한다
- 정상 기업 오판 가능성과 사기 미탐 가능성을 둘 다 고지한다

## 바이브 코딩용 체크리스트

- 새 기능이 생겨도 `위험도 예측 + 근거 설명 + 행동 권고` 구조를 유지하는가
- 분석 결과가 사용자가 이해할 수 있는 문장으로 나오는가
- 관리자 화면에서 룰과 경고 문구를 쉽게 바꿀 수 있는가
- 저장 데이터에 입력 유형, 요약, 시각, 점수, 근거가 남는가
- 법적 확정 표현 대신 참고용 예측 표현을 유지하는가
